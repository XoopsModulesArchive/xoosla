Xoosla V1.0.0.0 Genesis RC1

The Xoosla Development Team is pleased to announce the release of Xoosla V1.0.0.0 Genesis RC1.

This is the very first release of Xoosla CMS. This CMS is a direct Fork of Xoops v2.5.4 and has some changes, fixes and some bugs/bugfixes.

Note: if you plan on using this Xoosla and already have Xoops installed then you will have to update the database to include the required updated database items. YOu will find this infomoartion on our forums.
